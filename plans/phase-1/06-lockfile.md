## Lockfile Format

The lockfile is a TOML file at workspace root named `loom.lock`. Structure:

```toml
[metadata]
loom_version = "0.1.0"
generated_at = "2026-03-03T14:00:00Z"

workspace_members = [
    "lib/axi_common",
    "lib/axi_async_fifo",
    "projects/radar_processor",
]

[[package]]
name = "acmecorp/axi_common"
version = "1.3.0"
source = "workspace:lib/axi_common"
checksum = "sha256:abc123..."

[[package]]
name = "acmecorp/axi_async_fifo"
version = "1.2.0"
source = "workspace:lib/axi_async_fifo"
checksum = "sha256:def456..."
dependencies = ["acmecorp/axi_common >=1.0.0"]
```

## Types

```rust
use std::path::{Path, PathBuf};
use std::collections::HashMap;
use serde::{Deserialize, Serialize};
use chrono::Utc;
use crate::error::LoomError;
use crate::resolve::resolver::ResolvedProject;

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Lockfile {
    pub metadata: LockfileMetadata,
    #[serde(rename = "package", default)]
    pub packages: Vec<LockedPackage>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct LockfileMetadata {
    pub loom_version: String,
    pub generated_at: String,  // ISO 8601
    pub workspace_members: Vec<String>,  // relative paths from workspace root
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct LockedPackage {
    pub name: String,           // "org/component"
    pub version: String,        // "1.2.0"
    pub source: String,         // "workspace:lib/axi_common"
    pub checksum: String,       // "sha256:<hex>"
    #[serde(default)]
    pub dependencies: Vec<String>,  // "org/name >=1.0.0"
}

#[derive(Debug, Clone, PartialEq)]
pub enum LockfileStatus {
    Valid,
    Stale(Vec<String>),  // list of reasons
    Missing,
}
```

## Core Functions

/// Generate a lockfile from a resolved project.
pub fn generate_lockfile(
    resolved: &ResolvedProject,
    member_paths: &[crate::resolve::workspace::MemberPath],
) -> Result<Lockfile, LoomError> {
    let workspace_members = member_paths.iter()
        .map(|m| {
            // Make relative to workspace root
            m.path.strip_prefix(&resolved.workspace_root)
                .map(|rel| rel.to_string_lossy().replace('\\', "/"))
                .unwrap_or_else(|_| m.path.to_string_lossy().into_owned())
        })
        .collect();

    let packages = resolved.resolved_components.iter()
        .map(|comp| {
            let source = format!(
                "workspace:{}",
                comp.source_path
                    .strip_prefix(&resolved.workspace_root)
                    .map(|rel| rel.to_string_lossy().replace('\\', "/"))
                    .unwrap_or_default()
            );
            let checksum = compute_component_checksum(&comp.source_path)?;
            let dependencies = comp.manifest.dependencies.iter()
                .map(|(name, spec)| format!("{} {}", name, spec.version_string()))
                .collect();

            Ok(LockedPackage {
                name: comp.manifest.component.name.clone(),
                version: comp.resolved_version.to_string(),
                source,
                checksum,
                dependencies,
            })
        })
        .collect::<Result<Vec<_>, LoomError>>()?;

    Ok(Lockfile {
        metadata: LockfileMetadata {
            loom_version: env!("CARGO_PKG_VERSION").to_string(),
            generated_at: Utc::now().to_rfc3339(),
            workspace_members,
        },
        packages,
    })
}

/// Write lockfile to disk (workspace root / "loom.lock").
pub fn write_lockfile(lockfile: &Lockfile, workspace_root: &Path) -> Result<(), LoomError> {
    let path = workspace_root.join("loom.lock");
    let content = toml::to_string_pretty(lockfile)
        .map_err(|e| LoomError::LockfileWrite { message: e.to_string() })?;

    // Prepend a header comment
    let content = format!(
        "# This file is auto-generated by Loom. Do not edit manually.\n\
         # Commit this file to version control.\n\n{}",
        content
    );

    std::fs::write(&path, content)
        .map_err(|e| LoomError::Io { path, source: e })
}

/// Load lockfile from workspace root.
pub fn load_lockfile(workspace_root: &Path) -> Result<Option<Lockfile>, LoomError> {
    let path = workspace_root.join("loom.lock");
    if !path.exists() { return Ok(None); }

    let content = std::fs::read_to_string(&path)
        .map_err(|e| LoomError::Io { path: path.clone(), source: e })?;

    // TOML natively supports # comments, so no stripping needed.
    toml::from_str::<Lockfile>(&content)
        .map(Some)
        .map_err(|e| LoomError::LockfileParse { message: e.to_string() })
}

/// Check if an existing lockfile is still valid.
/// Spec: stale when dependency graph content changes or workspace member set changes.
pub fn check_staleness(
    lockfile: &Lockfile,
    resolved: &ResolvedProject,
    current_members: &[crate::resolve::workspace::MemberPath],
) -> LockfileStatus {
    let mut reasons = Vec::new();

    // Check workspace member set
    let current_member_set: std::collections::HashSet<String> = current_members.iter()
        .map(|m| {
            m.path.strip_prefix(&resolved.workspace_root)
                .map(|rel| rel.to_string_lossy().replace('\\', "/"))
                .unwrap_or_default()
        })
        .collect();

    let locked_member_set: std::collections::HashSet<String> =
        lockfile.metadata.workspace_members.iter().cloned().collect();

    if current_member_set != locked_member_set {
        let added: Vec<_> = current_member_set.difference(&locked_member_set).collect();
        let removed: Vec<_> = locked_member_set.difference(&current_member_set).collect();
        if !added.is_empty() {
            reasons.push(format!("New workspace members: {:?}", added));
        }
        if !removed.is_empty() {
            reasons.push(format!("Removed workspace members: {:?}", removed));
        }
    }

    // Check resolved packages match lockfile
    let locked_packages: HashMap<&str, &LockedPackage> = lockfile.packages.iter()
        .map(|p| (p.name.as_str(), p))
        .collect();

    for comp in &resolved.resolved_components {
        let name = &comp.manifest.component.name;
        match locked_packages.get(name.as_str()) {
            None => reasons.push(format!("New dependency: {}", name)),
            Some(locked) => {
                if locked.version != comp.resolved_version.to_string() {
                    reasons.push(format!(
                        "Version changed for {}: {} → {}",
                        name, locked.version, comp.resolved_version
                    ));
                }
            }
        }
    }

    // Check for removed packages
    for locked_name in locked_packages.keys() {
        if !resolved.resolved_components.iter().any(|c| c.manifest.component.name == *locked_name) {
            reasons.push(format!("Removed dependency: {}", locked_name));
        }
    }

    if reasons.is_empty() {
        LockfileStatus::Valid
    } else {
        LockfileStatus::Stale(reasons)
    }
}

/// Compute a SHA-256 checksum of all files in a component directory
/// (component.toml + all fileset files).
fn compute_component_checksum(component_dir: &Path) -> Result<String, LoomError> {
    use sha2::{Sha256, Digest};

    let mut hasher = Sha256::new();

    // Hash component.toml
    let manifest_path = component_dir.join("component.toml");
    let content = std::fs::read(&manifest_path)
        .map_err(|e| LoomError::Io { path: manifest_path, source: e })?;
    hasher.update(&content);

    // We hash the manifest only for Phase 1 — not all source files.
    // Source file changes are tracked by the build cache, not the lockfile.

    let result = hasher.finalize();
    Ok(format!("sha256:{}", hex::encode(result)))
}

## Integration into Build Pipeline

In the build pipeline (Phase 1), the flow is:
1. Load lockfile from workspace root (`load_lockfile`)
2. If missing: run resolution, generate lockfile, write it, proceed
3. If present: check staleness (`check_staleness`)
   - If stale: error with "Lockfile is stale. Run `loom deps update` to regenerate."
   - If valid: use lockfile's resolved versions (for now, just proceed — Phase 7 registry will use pinned versions)
